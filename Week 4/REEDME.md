# Week 4 — Advanced SOC Operations Practical

## Overview

This folder contains documentation and artifacts for Week 4, focusing on advanced Security Operations Center (SOC) activities including threat hunting, SOAR automation, adversary emulation, incident response, and performance evaluation.

The practical was conducted in an authorized lab environment using multiple industry-standard security tools.

---

## Objectives

- Detect advanced threats using hypothesis-driven hunting
- Automate incident response using SOAR playbooks
- Perform adversary emulation
- Analyze alerts using threat intelligence
- Preserve and analyze digital evidence
- Measure SOC performance metrics
