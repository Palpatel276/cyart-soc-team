# SOC Monitoring Workflow

## Step 1 – Environment Setup
Install Wazuh SIEM and configure virtual machines.

## Step 2 – Alert Monitoring
Access Wazuh Threat Hunting dashboard and monitor security events.

## Step 3 – Attack Simulation
Use Metasploit to exploit VSFTPD vulnerability on Metasploitable2.

## Step 4 – Detection
Wazuh detects exploit attempt and generates security alert.

## Step 5 – Alert Triage
Analyze alert details and identify suspicious activity.

## Step 6 – Threat Intelligence
Validate indicators using VirusTotal and AlienVault OTX.

## Step 7 – Incident Response
Isolate system and block attacker IP using CrowdSec.

## Step 8 – Evidence Preservation
Collect memory dump and record SHA256 hash.

## Step 9 – Reporting
Document the investigation and prepare SOC incident report.
