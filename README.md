## Week 2 — Alert Management & Incident Response Fundamentals

### Theoretical Focus

- Alert priority levels and CVSS scoring
- Incident classification frameworks (MITRE ATT&CK, ENISA, VERIS)
- Incident response lifecycle (NIST SP 800-61)

### Practical Activities

- Alert classification and prioritization
- Incident ticket creation in TheHive
- Alert triage using threat intelligence
- Evidence preservation and chain of custody
- Full alert-to-response cycle simulation

### Key Skills Developed

- Incident handling procedures
- Alert prioritization
- Documentation and reporting
- Communication and escalation
