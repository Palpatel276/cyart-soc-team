# Week 1 Notes — SOC Fundamentals

## SOC Fundamentals
- Purpose: Threat detection, incident response, monitoring
- Roles: Tier 1, Tier 2, Tier 3 analysts, SOC manager
- Frameworks: NIST, MITRE ATT&CK

## Security Monitoring
- Detect anomalies and unauthorized access
- Tools: SIEM, Wireshark
- Metrics: False positives, MTTD

## Log Management
- Collection → Normalization → Storage → Analysis
- Common logs: Windows Event Logs, Syslog, HTTP logs

## Security Tools
- SIEM: Splunk, Elastic, Wazuh
- IDS/IPS: Snort
- Vulnerability Scanner: Nessus
- Endpoint Monitoring: Osquery

## Basic Concepts
- CIA Triad
- Risk, Threat, Vulnerability
- Defense-in-depth, Zero Trust

## Incident Response
Preparation → Identification → Containment → Eradication → Recovery → Lessons Learned

## Documentation
- Incident reports
- Runbooks
- SOPs
- Post-incident analysis
